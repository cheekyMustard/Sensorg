# 📐 Wireframes Übersicht

Dieses Paket enthält die ersten Wireframes für die **Sensational Organiser App**.  
Alle Wireframes sind in Grau gehalten und dienen als visuelle Orientierung für die spätere Umsetzung im Frontend.

## Enthaltene Wireframes

1. **home_wireframe.png**
   - Delivery Requests (Übersicht nach Datum)
   - Notes Vorschau (rollen- und prioritätsabhängig)
   - What else can be done (tägliche/wiederkehrende Aufgaben)
   - Nice to know (Tipps, Infos)
   - Witz des Tages

2. **notes_wireframe.png**
   - Notes Übersicht
   - Filter (Open, In Progress, Done)
   - Archivfunktion (für Admins sichtbar)

3. **requests_wireframe.png**
   - Delivery Requests Liste
   - Filter nach Status
   - Kalenderansicht (im Browser)

4. **profile_wireframe.png**
   - Benutzerprofil (Name, Rolle)
   - Admin-Optionen (User anlegen, Rollen verwalten, Passwort zurücksetzen)
   - Witze-Sammlung (Texte oder Bilder)
   - Nice to know Erstellung (Admin)

## Nutzung

Diese Wireframes sind als **erste visuelle Grundlage** gedacht.  
Sie können während der Entwicklung erweitert oder angepasst werden.
